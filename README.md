# Create Customer Segment
Neste projeto, foi aplicado técnicas de aprendizagem não supervisionada em dados sobre gastos, coletados de clientes de uma distribuidora atacadista em Lisboa, para identificar segmentos de clientes ocultos nos dados. Primeiro, você irá explorar os dados selecionando um pequeno subconjunto como amostra e determinar se alguma das categorias de produtos está altamente correlacionada com outra. Depois, pré-processará os dados, dimensionando cada categoria de produto e identificando (e removendo) valores aberrantes. De posse dos dados "limpos", você aplicará PCA a eles e implementará os algoritmos de clustering para criar os segmentos. Por último, irá comparar a segmentação encontrada com uma marcação adicional e considerar a maneira como essa informação poderia auxiliar a distribuidora atacadista com futuras mudanças de serviço.
